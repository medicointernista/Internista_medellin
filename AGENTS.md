## Project Summary
A professional, modern medical website for Dr. Willmer Obregón, an Internal Medicine specialist in Medellín. The project features a clean "Medical Modern" aesthetic with soft teals, light blues, glassmorphism effects, and high-quality photography.

## Tech Stack
- Next.js 15
- React 19
- Tailwind CSS 4
- Lucide React (Icons)
- TypeScript

## Architecture
- `src/app/`: Next.js App Router files (layout, page, globals.css)
- `src/components/sections/`: Major UI sections (Header, Hero, Services, About Me, etc.)
- `src/components/ui/`: Reusable UI components (StickySocialSidebar)
- `public/`: Static assets (images, logos)

## User Preferences
- Language: Spanish (es)
- Theme: Light mode (medical aesthetic)
- Design: Modern, clinical, welcoming, generous whitespace, rounded corners (large radii)

## Project Guidelines
- Use functional components with TypeScript
- Follow the "Medical Modern" brand art direction
- Responsive design (mobile-first)
- Use Lucide icons for medical and contact representations

## Common Patterns
- Section-based page construction
- Glassmorphism for headers and cards
- Primary teal gradient for CTAs
